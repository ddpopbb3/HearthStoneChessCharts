## ✨ 特性

* 统计了各个种类卡牌的数量(柱状图)(饼状图)
* 统计了各个星级卡牌的数量(柱状图)

## 📝 使用
* python HearthStoneChessCharts\HearthStoneChess.py
* 会在生成战棋目录, 存放有数据json
* 会在当前目录下生成render.html, 可以预览图表
